# **App Name**: TestiAI

## Core Features:

- Input Product/Service Details: Allows users to input comprehensive information about their product or service.
- AI Testimonial Draft Generation: Generates realistic, human-like testimonial drafts using VentixlabsAI tool based on input details.
- Customization Options: Allows users to customize tone, length, and other attributes of the generated testimonial.
- Draft Preview: Provides a real-time preview of the generated testimonial drafts.
- Copy to Clipboard: Enables users to quickly copy the generated testimonial to their clipboard.

## Style Guidelines:

- Primary color: Bright red (#FF0000) to capture attention and align with the user's branding request.
- Background color: Black (#141414) to provide a stark contrast, in an apple.com style, emphasizing content and calls to action.
- Accent color: Analogous to red is orange; use a light-ish orange (#FF7700) for interactive elements like buttons, ensuring good contrast against the dark background.
- Font: 'Inter', a grotesque sans-serif with a neutral and modern look, will be used for both headlines and body text to maintain simplicity and readability.
- Adopts an apple.com-like interface with a clean, minimalist design focused on user experience and content presentation.
- Simple, outline-style icons for UI elements to complement the minimalist design.
- Subtle transitions and animations for a smooth and engaging user experience, without being distracting.
